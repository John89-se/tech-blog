良い道具を使うことは、プログラミングにおいて非常に重要です。
世界で最も使われているエディタ **VS Code** に入れるべき拡張機能を厳選しました。

## 1. ESLint / Prettier

コードの整形ヤクザです。
`Ctrl + S` で保存するたびに、インデントのズレやセミコロンの有無を自動で直してくれます。
これ無しでコードを書くのは、定規無しで線を引くようなものです。

## 2. Tailwind CSS IntelliSense

Tailwindを使うなら必須。
クラス名の補完はもちろん、現在の色がどんな色か、四角いアイコンで表示してくれます。

## 3. GitLens

「この行、誰がいつ書いたの？」
コードの行末に、最後のコミット情報が薄く表示されます。
バグの原因調査や、過去の自分のコードへの絶望に役立ちます。

## 4. GitHub Copilot

一度使ったら戻れないAIアシスタント。
コメントで「日付をフォーマットする関数」と書けば、実装を提案してくれます。
月額料金以上の価値は間違いなくあります。

## 5. Material Icon Theme

フォルダやファイルのアイコンが見やすくなります。
見た目が良いと、モチベーションも上がりますよね。

## まとめ

環境構築に時間をかけすぎるのは良くないですが、これらは最初に入れておいて損はありません。
快適なコーディングライフを！
