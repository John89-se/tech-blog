プレーンテキストで書ける、軽量なマークアップ言語です。
GitHubのREADMEや、このブログの記事も全てMarkdownで書かれています。

## よく使う記法

### 見出し (Headers)
`#` の数で大きさが決まります。
```md

## H2 (大見出し)
### H3 (中見出し)
```

### リスト (Lists)
ハイフン `-` か 数字 `1.` でリストになります。
```md
- りんご
- みかん

1. 手順1
2. 手順2
```

### リンクと画像
```md
[Google](https://google.com)
![代替テキスト](/image.png)
```
リンクは `[表示文字](URL)`、画像は先頭に `!` をつけるだけ。

### コードブロック
バッククォート3つ ` ``` ` で囲むとコードになります。
言語を指定するとシンタックスハイライトも効きます。

## なぜMarkdown？

*   **どこでも使える**: エディタを選ばない。
*   **読みやすい**: ソースコードのままでも読める。
*   **変換しやすい**: HTMLやPDFに簡単に変換できる。

今日からメモはMarkdownで取りましょう！
