# Webサイトは公開して初めて完成する

一生懸命作ったポートフォリオやブログ、自分のPCの中に眠っていませんか？
**Vercel** を使えば、クレジットカード登録なしで、SSL（https）対応のサイトを公開できます。

## Vercelとは？

Next.jsの開発チームが作ったホスティングサービスです。
Next.jsのために最適化されていますが、普通のHTMLサイトやReactアプリも公開できます。

## デプロイの手順（GitHub連携編）

一番オーソドックスな方法です。

1.  GitHubにソースコードをPushする
2.  Vercelのダッシュボードで「New Project」をクリック
3.  GitHubのリポジトリを選択して「Deploy」

これだけです。
あとはGitHubにコードを修正してPushするたびに、Vercelが自動で検知して本番環境を更新してくれます（CI/CD）。

## デプロイの手順（CLI編）

Gitを使わなくても、ターミナルからコマンド一発で公開できます。

```bash
npm i -g vercel
vercel
```

いくつかの質問に答えるだけで、その場のコードがアップロードされます。
手軽にテストしたい時はこちらが便利です。

## カスタムドメインも無料

`my-site.vercel.app` というドメインがもらえますが、自分で買ったドメイン（例: `my-name.com`）を割り当てることも簡単です。
これでお金がかからないのは驚異的です。

## まとめ

「サーバー契約」「FTPアップロード」…そんな時代は終わりました。
Vercelを使って、あなたの作品を世界に見てもらいましょう！
