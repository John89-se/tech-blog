# オフィスに行かなくていい時代の働き方

リモートワークが当たり前になりましたが、環境は整っていますか？
快適に働くためのツールを紹介します。

## 1. Notion (情報管理)

ドキュメント、タスク管理、Wiki、全てこれ一つで完結します。
チーム全員が同じ情報にアクセスできる「シングル・ソース・オブ・トゥルース」を作ることが重要です。

## 2. Raycast (ランチャー)

MacユーザーならAlfredの代わりにRaycastをおすすめします。
計算、クリップボード履歴、Window操作、ChatGPT呼び出し…
キーボードから手を離さずに何でもできます。

## 3. Miro (オンラインホワイトボード)

図解やマインドマップを書きながら話したい時、Zoomの画面共有だけでは限界があります。
Miroなら、無限に広がるキャンバスに全員で付箋を貼れます。

## 4. Krisp (ノイズキャンセリング)

カフェや工事現場の近くで会議しても、相手にはあなたの声しか聞こえません。
AIが背景ノイズを完全に消してくれます。これがあれば場所を選ばず働けます。

## まとめ

ツールは手段ですが、良いツールは「仕事のストレス」を劇的に減らしてくれます。
気になったものがあれば試してみてください。
